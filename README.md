# Futsal Finder

Futsal Finder is a Flutter application that connects futsal players with futsal owners. It allows users to browse and book available futsals, and provides a platform for futsal owners to manage their listings and bookings.

## Features

- **User Authentication:** Users can sign up, sign in, and sign out of the application.
- **Role-based Access Control:** The application has three user roles: User, Futsal Owner, and Admin.
  - **Users:** Can browse and book available futsals.
  - **Futsal Owners:** Can add, edit, and delete their futsals. They can also view bookings for their futsals.
  - **Admins:** Can manage users and futsals.
- **Futsal Listings:** Futsal owners can list their futsals with details such as name, location, price, description, amenities, and images.
- **Search and Filter:** Users can search for futsals by name and filter them by location, price, and availability.
- **Booking System:** Users can book a futsal for a specific date and time.
- **Admin Dashboard:** Admins have access to a dashboard where they can manage users and futsals.
- **Futsal Owner Dashboard:** Futsal owners have access to a dashboard where they can manage their futsals and bookings.

## Technologies Used

- **Flutter:** The application is built using the Flutter framework.
- **Firebase:** The application uses Firebase for authentication, database, and storage.
  - **Firebase Authentication:** For user authentication.
  - **Cloud Firestore:** As the database for storing user and futsal data.
  - **Firebase Storage:** For storing futsal images.
- **Provider:** For state management.
- **Google Fonts:** For custom fonts.
- **Image Picker:** For picking images from the gallery or camera.

## Getting Started

To get started with the project, you will need to have Flutter and Firebase set up on your local machine. You can then clone the repository and run the app on your emulator or device.

```
flutter run
```
